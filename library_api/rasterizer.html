Rasterizer#

Splatwizard integrates multiple rasterizers from prior work, which can be directly imported and used. Each rasterizer closely follows the design in the original codebase to minimize tedious migration work.

Standard Gaussian rasterizer#

class splatwizard.rasterizer.gaussian.GaussianRasterizer(raster_settings: GaussianRasterizationSettings)

Standard Gaussian Splatting rasterizer from 3D Gaussian Splatting for Real-Time Radiance Field Rendering.

The raster_setting class is defined as

class splatwizard.rasterizer.gaussian.GaussianRasterizationSettings(
    image_height: int,
    image_width: int,
    tanfovx: float,
    tanfovy: float,
    bg: torch.Tensor,
    scale_modifier: float,
    viewmatrix: torch.Tensor,
    projmatrix: torch.Tensor,
    sh_degree: int,
    campos: torch.Tensor,
    prefiltered: bool,
    debug: bool
)

Rasterizer function is defined as

class GaussianRasterizer(nn.Module):
    ...
    def rasterizer(self,
        means3D: Tensor, 
        means2D: Tensor, 
        opacities: Tensor, 
        shs: Tensor | None, 
        colors_precomp: Tensor | None, 
        scales: Tensor | None, 
        rotations: Tensor| None, 
        cov3D_precomp: Tensor | None
    ):
        ...
        return rendered_image, radii

FlashGS#
